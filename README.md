# BACustomAlertView
一个完全实现自定义的alertView！

本alert特点：<br />
1、手势触摸隐藏<br />
2、可以自定义背景图片、按钮颜色<br />
3、可以添加文字和图片，且可以滑动查看<br />
4、完全兼容目前所有 iOS 系统版本

####注：目前此版本只支持竖屏适配，小伙伴儿记得哦！
